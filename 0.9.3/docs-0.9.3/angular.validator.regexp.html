angular.validator.regexp

Description

Use regexp validator to restrict the input to any Regular Expression.

Usage

In HTML Template Binding

<input type="text" ng:validate="regexp:expression"/>

In JavaScript

angular.validator.regexp(value, expression);

Parameters

  • value:string: value to validate
  • expression:regexp: regular expression.

CSS

ng-validation-error

Example

Enter valid SSN: <input name="ssn" value="123-45-6789" ng:validate="regexp:/^\d\d\d-\d\d-\d\d\d\d$/" > it('should invalidate non ssn', function(){ var textBox = element('.doc-example :input'); expect(textBox.attr('className')).not().toMatch(/ng-validation-error/); expect(textBox.val()).toEqual('123-45-6789'); input('ssn').enter('123-45-67890'); expect(textBox.attr('className')).toMatch(/ng-validation-error/); });
