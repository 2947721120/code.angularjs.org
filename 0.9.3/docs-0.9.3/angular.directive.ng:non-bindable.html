angular.directive.ng:non-bindable

Description

Sometimes it is necessary to write code which looks like bindings but which should be left alone by <angular/>. Use ng:non-bindable to ignore a chunk of HTML.

Usage

In HTML Template Binding

<ANY ng:non-bindable="ignore">
  ...
</ANY>

Parameters

  • ignore:string:

    Sometimes it is necessary to write code which looks like bindings but which should be left alone by <angular/>. Use ng:non-bindable to ignore a chunk of HTML.

Example

In this example there are two location where is present, but the one wrapped in ng:non-bindable is left alone

<div>Normal: {{1 + 2}}</div> <div ng:non-bindable>Ignored: {{1 + 2}}</div> it('should check ng:non-bindable', function(){ expect(using('.doc-example-live').binding('1 + 2')).toBe('3'); expect(using('.doc-example-live').element('div:last').text()). toMatch(/1 \+ 2/); });
